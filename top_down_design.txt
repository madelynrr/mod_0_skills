How to Become a Turing Student

1 Apply to Turing
  1.1 Fill out online application
    1.1.1 Sign in through facebook or github
      1.1.1.1 Fill out general application
        1.1.1.1.1 Enter name
        1.1.1.1.2 Enter email address
        1.1.1.1.3 Enter address
    1.1.2 Submit resume
      1.1.2.1 Edit resume
      1.1.2.2 Upload resume
    1.1.3 Complete logic problem test
      1.1.3.1 Set timer
      1.1.3.2 Open logic problem test
      1.1.3.3 Complete each logic problem
      1.1.3.4 Submit test
2 Interview at Turing
  2.1 Set time to Interview
  2.2 Prepare for interview
    2.2.1 Pick out interview clothes
    2.2.2 Practice logic problems in front of people/explaining to others
3 Go to Try Coding Weekend
  3.1 Sign up for Try Coding Weekend
  3.2 Attend Try Coding Weekend
  3.3 Decide which track to do
4 Complete new student assignments
  4.1 Pay tuition
  4.2 Complete intake survey
  4.3 Sign up for Mod 0
5 Complete Mod 0
  5.1 Attend all Mod 0 Sessions
    5.1.1 Complete all homework assignments
  5.2 Complete Mod 0 Capstone
